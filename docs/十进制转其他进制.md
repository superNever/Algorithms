## 原理
十进制N和其他d进制的转换原理为：

```bash
N = (N div d) * d+ N mode d (其中div为整除运算，mod为求余运算)
```
例如：(1348) 转为8进制

N | N div 8 | N mod 8
---|---|---
1348 | 168 | 4
168 | 21 | 0
21| 2| 5
2|0|2

```bash

const conversion = (data, num) => {
    let arr = [];
    let d = data;
    let str = '';
    while (d > 0) {
        arr.push(d%num)
        d = Math.floor(d/num)
    }
    for (let i =0,j = arr.length; i<j; i++){

    }

}
```
